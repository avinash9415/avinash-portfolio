# 📖 Project Overview

This is my **Personal Portfolio Website** developed using **HTML, CSS, and JavaScript** as part of my internship/project assignment.

The website showcases my profile, technical skills, projects, education, and contact information through a modern, responsive, and interactive interface. It is designed to provide recruiters and visitors with a clear overview of my work and capabilities.

- Responsive Design
